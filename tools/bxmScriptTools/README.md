# Instructions

Drop bxm (.bxm, .sar, .gad, ...) or xml files onto either `bxmToXml.py` or `xmlToBxm.py` to convert to XML or BXM.

Or via command line `python [xToX.py] [input] [output (optional)]`.

And of course you need python (3.X) installed.
